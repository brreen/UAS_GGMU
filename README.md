1.pip install pandas requests torch matplotlib seaborn beautifulsoup4 selenium nltk transformers python-dotenv

2.IG_USER="username_instagram"
   IG_PASS="password_instagram"
   HF_TOKEN="token_huggingface"

3.Setelah skrip berjalan, program akan melakukan scraping artikel berita terlebih dahulu.

4.Selanjutnya, jendela browser otomatis (Selenium) akan terbuka dan menuju halaman login Instagram.

5.Kode akan mencoba mengisi username dan password secara otomatis menggunakan metode simulasi ketikan manusia (Action Chains).

6.Perhatikan Layar Browser:

7.JIKA muncul verifikasi keamanan atau Captcha, silakan isi dan selesaikan secara manual terlebih dahulu di jendela browser tersebut.
JIKA TIDAK muncul Captcha dan akun Anda langsung berhasil masuk ke halaman Beranda/Homepage Instagram, berarti Anda tidak perlu mengisi apa-apa.

8.Setelah posisi browser Anda sudah benar-benar berada di halaman Beranda/Homepage Instagram, kembali ke Terminal/CMD Anda, lalu tekan tombol ENTER untuk melanjutkan proses pencarian 33 konten secara otomatis.

9.Di akhir proses, program akan memunculkan jendela pop-up grafik visualisasi data (Dashboard dan Grafik Perbandingan).

10.Langkah Terakhir: Jika grafik sudah selesai dilihat atau ingin disimpan, cukup klik tombol silang (X) di pojok kanan/kiri atas jendela grafik tersebut untuk menutupnya.

11.Setelah grafik ditutup, terminal akan menampilkan pesan === PIPELINE SELESAI SEMPURNA === dan proses selesai sepenuhnya.
